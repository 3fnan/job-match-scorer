# Job Match Scorer

An NLP-powered web app that scores how well your resume matches a job description.

## What it does
- Upload your resume as PDF
- Paste any job description
- Get a match score (0-100%) instantly
- See missing keywords to improve your resume

## Tech Stack
Python, FastAPI, Scikit-learn, pdfplumber, HTML/CSS

## How to Run
```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
Open http://127.0.0.1:8000
